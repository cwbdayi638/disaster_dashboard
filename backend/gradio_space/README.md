# 即時防災資訊智慧展示 (Visual)

此空間為即時防災資訊整合系統的視覺展示介面。它會即時從 GitHub Repository 抓取最新更新的災情資料，並透過動態圖表呈現統計結果與提供列表查詢。

- **資料來源**: [GitHub Repository](https://github.com/cwbdayi638/disaster_dashboard)
- **架構**: Python + Gradio + Plotly
